⚡ Quick Start Guide

Get up and running with the Blaze Intelligence API in under 5 minutes.

1. Get Your API Key

Sign up and generate your API key from the developer portal.

# Your API key will look like this:
export BLAZE_API_KEY="bz_live_1234567890abcdef"

2. Make Your First API Call

cURL
JavaScript
Python
curl -X GET "https://api.blaze-intelligence.com/v1/analytics/cardinals/latest" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
const response = await fetch('https://api.blaze-intelligence.com/v1/analytics/cardinals/latest', {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
});

const data = await response.json();
console.log(data);
import requests

headers = {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
}

response = requests.get(
    'https://api.blaze-intelligence.com/v1/analytics/cardinals/latest',
    headers=headers
)

data = response.json()
print(data)

🔐 Authentication

Blaze Intelligence API uses API keys for authentication. Include your API key in the Authorization header of every request.

API Key Format

Authorization: Bearer bz_live_1234567890abcdef

Environment Types

  • Sandbox: bz_test_ - For development and testing
  • Production: bz_live_ - For live applications

📊 Analytics API

Access comprehensive sports analytics and performance metrics.

Team Analytics

GET

/v1/analytics/{team}/latest

Get latest team performance metrics and readiness scores.

Player Performance

GET

/v1/analytics/players/{playerId}

Individual player statistics and performance trends.

Game Analytics

GET

/v1/analytics/games/{gameId}

Detailed game statistics and real-time metrics.

Example: Get Cardinals Latest Analytics

const response = await fetch('https://api.blaze-intelligence.com/v1/analytics/cardinals/latest', {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY'
  }
});

const analytics = await response.json();

// Access readiness scores
console.log(`Overall Readiness: ${analytics.data.readiness.overall}%`);
console.log(`Offense Rating: ${analytics.data.readiness.offense}%`);
console.log(`Pitching Rating: ${analytics.data.readiness.pitching}%`);
